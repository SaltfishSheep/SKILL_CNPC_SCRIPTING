## Notes

- 1.7.10 through 1.18.2 share the same doc structure on `kodevelopment.nl`.
- 1.20.1+ uses a completely different documentation site (GitHub Pages), maintained by the community.
  Goodbird's port supports 1.19.2, 1.20.1, 1.20.2, 1.20.4, 1.20.6, 1.21, 1.21.1 on Forge/Fabric/NeoForge.
- For versions between those listed, use the closest match. If uncertain, ask the user before fetching.
- All kodevelopment.nl docs are JavaDoc-style. Navigate by appending package/class paths (e.g., `noppes/npcs/...`).
- **CustomNPC+** (1.7.10 fork by KAMKEEL): uses **cur** conventions. See `SKILL.md` Workflow §1.
- **BetaZavr fork** (1.12.2 unofficial): adds deobfuscation, `dump()`, client-side scripts.
  Also has experimental 1.20.1. Always ask 1.12.2 users which version they use.
