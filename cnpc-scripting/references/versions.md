If the user's exact minor version isn't listed, pick the nearest match (e.g., 1.12.0 → use 1.12.2 docs).

## Notes

- 1.7.10 through 1.18.2 share the same doc structure on `kodevelopment.nl`.
- 1.20.1+ uses a completely different documentation site (GitHub Pages), maintained by the community.
- For versions between those listed, use the closest match. If uncertain, ask the user before fetching.
- All kodevelopment.nl docs are JavaDoc-style. Navigate by appending package/class paths (e.g., `noppes/npcs/...`).
- **CustomNPC+** is an unofficial fork by KAMKEEL for 1.7.10 with extended APIs and 1.8+-style
  script structure. Always ask the user if they're on CustomNPC+ when they say 1.7.10.
- **BetaZavr fork** is an unofficial 1.12.2 branch with extended features (built-in
  deobfuscation table, `dump()` debug method, client-side scripting). Always ask 1.12.2
  users which version they use.
